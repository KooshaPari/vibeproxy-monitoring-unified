# Code of Conduct

This repository follows the Phenotype organization conduct baseline.

## Expected Behavior

- Be respectful and constructive in issues, pull requests, and reviews.
- Keep discussions focused on the repository scope and evidence.
- Assume good intent while responding directly to technical risk.

## Unacceptable Behavior

- Harassment, threats, or discriminatory language.
- Publishing private information without explicit permission.
- Disruptive comments that derail technical work without evidence.

## Reporting

Report conduct concerns to the repository owner or organization maintainers.
Security-sensitive reports should follow `SECURITY.md`.
